В папке files находятся: файл с текстом для тестирования (source.txt) и файл куда сохраняется результат (result.txt). Пути по умолчанию указывают на них.

Программа предусматривает 4 вида сортировки:
1. Сортировка по количеству во возрастанию (для одинакового количества сортируются по алфавиту по возрастанию).
2. Сортировка по количеству во убыванию (для одинакового количества сортируются по алфавиту по возрастанию).
3. Сортировка по алфавиту по возрастанию.
4. Сортировка по алфавиту по убыванию.
